Software Requirements Specification (SRS) 
ResearchRAG AI – Intelligent Academic Research Assistant 
1. Project Name & Background  
 
Project Name: ResearchRAG AI 
 Background: Researchers face difficulty finding relevant papers, classifying topics, and 
extracting insights efficiently. Traditional keyword search lacks semantic understanding 
and citation grounding. 
 Problem Statement: Current tools lack an integrated solution for combining semantic 
search, automatic classification, and citation-grounded question answering in one system. 
2. Objectives: 
a. Paper Classification: Title + abstract → category prediction (ML models) 
b. Research Q&A (RAG): Retrieve papers → citation-based answers (Gemini API) 
c. Semantic Search: Embedding-based similarity search 
d. Fallback Mode: Local predictions + retrieved context if API unavailable 
e. Interface: Streamlit web app
